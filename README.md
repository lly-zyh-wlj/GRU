# 20Newsgroups 二分类 GRU 模型实现

小组成员：刘凌羽，张溢慧，吴丽君
## 项目任务
以 20newsgroups (解压20news-bydate.rar所得)数据中的 `alt.atheism` 和 `soc.religion.christian` 两类为二分类样本，使用 **1 层 GRU 模型**完成文本分类任务，要求测试集准确率 > 0.7。

## 数据处理

### 数据划分
- 原始训练集按 **0.8 / 0.2** 划分为训练集和验证集
- 测试集保持不变

### 文本预处理
- 去除 HTML 标签、邮件头、标点、数字
- 小写化、分词、去除英文停用词和噪声词
- **仅用训练集构建词汇表**，避免数据泄露
- 序列长度设置为 **300**，保留完整语义

### 类别平衡
- 采用**分层抽样**划分训练/验证集，保证类别分布一致

## 模型结构
- **嵌入层**：支持预训练 GloVe 词向量，随机初始化时添加 Dropout 正则
- **GRU 层**：1 层**双向 GRU**，捕捉上下文语义，输出维度 256
- **分类层**：LayerNorm + Dropout + 全连接层，增强泛化性
- **正则化**：Dropout(0.6)、LayerNorm、AdamW 权重衰减

## 超参数选择

| 超参数 | 取值 | 选择理由 |
|--------|------|----------|
| 嵌入维度 | 128 | 平衡语义表达与计算效率 |
| GRU 隐藏层维度 | 256 | 足够捕捉文本特征，避免过拟合 |
| Dropout | 0.6 | 有效抑制过拟合 |
| 学习率 | 0.0005 | 适配 AdamW 优化器，避免训练震荡 |
| 权重衰减 | 1e-4 | 抑制模型复杂度，提升泛化性 |
| 批次大小 | 32 | 小批次提升泛化性，适配小样本量 |
| 最大序列长度 | 300 | 保留完整文本语义，避免截断 |

## 优化方法
- **优化器**：AdamW，比 Adam 更优的权重衰减策略，缓解过拟合
- **学习率调度**：余弦退火调度，动态调整学习率，提升收敛稳定性
- **早停策略**：patience=5，验证集准确率不提升时提前终止，避免过拟合
- **双向 GRU**：捕捉上下文双向语义，提升特征提取能力
- **预训练词向量**：使用 GloVe 预训练嵌入，大幅提升小数据集泛化性

## 实验结果
- 验证集最佳准确率：**~0.75**
- 测试集准确率：**~0.73**（满足 >0.7 的要求）
- 训练日志：无严重过拟合，训练/验证损失同步收敛
